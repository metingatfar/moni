# Widgets Report

## Widget Library
Specifies metadata and rendering options for reusable dashboard widgets:
- **Project Card**: Renders project name, last edited timestamp, and technology tags.
- **Statistics Ring**: Renders dynamic progress percentages (e.g. build success).
- **Risk Meter**: Renders gauge charts mapping vendor lock-in and migration indexes.
- **Provider Widget**: Summarizes active LLM selections.

- **Status**: Reusable components declared.
