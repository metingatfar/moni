# Visual Builder Report

* **Overview**: The Visual Builder transforms MONI Studio into a responsive application visual designer.
* **Component Models**: Drag-and-drop structures bind to design system tokens with responsive layout constraints.
* **Master Controller**: Managed by `VisualBuilderEngine` which coordinates alignment checks, hierarchical nodes, state history, preview scales, and export planning schedules.
* **Safety & Security**: Operates strictly in mock/proposal mode to safeguard existing workspace files.
