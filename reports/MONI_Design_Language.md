# MONI Design Language (MDL)

MDL sets design standards to ensure the user interface remains spacious, minimal, and premium.

## Core Rules

1. **VisionOS Depth**:
   - Translucent container designs using `backdrop-filter: blur(20px)`.
   - Thin `border: 1px solid rgba(255, 255, 255, 0.06)` borders reflecting specular glare.

2. **Apple Typography**:
   - Clean, highly legible `Inter` or native sans-serif fonts.
   - Letter-spacing (`letter-spacing: 0.05em`) applied to headers to improve scannability.

3. **Palette Tokens**:
   - Deep navy bases (`#030712`, `#0b0f19`) to keep contrast easy on the eyes.
   - Glowing purple/cyan accents to represent active artificial intelligence components.
