# MONI 2.0 Mobile Experience & Navigation Report

## 1. Architecture
* **Responsive Breakpoint:** Triggered when screen width is under `768px`.
* **Bottom Navigation Bar:** Dedicated bottom tab list (Home, Chat, Tasks, Memory, Settings).
* **Mobile Header:** Minimalist 56px header with quick action switches.

## 2. Views
* **Home:** Contains greetings, active reminders, system widgets, and the main interactive assistant Orb.
* **Assistant Screen:** Full screen interactive mic mode with canvas/wave overlays.
