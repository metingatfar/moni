# Folder Structure Report

## Standard Layout Patterns
The `FolderStructureGenerator` structures paths based on architectural style conventions:
- **Clean Architecture Layout**: Segregates layers into `core/`, `domain/`, `data/`, and `presentation/`.
- **Feature-first Layout**: Maps folders directly to features (`features/login/components`, `features/login/api`).
- **Standard Backend Layout**: Maps `api/controllers`, `api/middleware`, `core/config`, and `tests/`.

- **Output State**: Folder array structures mapped in-memory.
- **Precision Score**: 95%
