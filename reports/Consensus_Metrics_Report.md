# Consensus Metrics Report

* **Engine**: ConsensusMetrics
* **Indicators Tracked**:
  - Total consensus executions.
  - Average agreement and disagreement ratios.
  - Saptanan Çakışmalar (Conflict count).
  - Average confidence.
  - Average reasoning score.
  - Per-provider win rate matrix.
* **Diagnostics Status**: Connected directly to the `MoniDashboard` UI state provider.
