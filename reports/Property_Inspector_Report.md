# Property Inspector Report

* **Overview**: Edits specific styling values, text labels, visibility switches, and accessibility options.
* **Token Mappings**: Forces binding properties directly to DesignSystem token tags (e.g. `PrimaryColor`, `SpacingLarge`) instead of using hardcoded styling.
* **Metadata Export**: Automatically translates token bindings to CSS variable outputs (`var(--token-PrimaryColor)`).
