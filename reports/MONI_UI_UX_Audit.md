# MONI 2.0 UI/UX Visual & Accessibility Audit

## 1. Review Summary
* **Contrast:** Satisfies WCAG AA contrast rules for readability over dark background.
* **Interactive Scales:** Micro-animations scale down on click, giving responsive physical feedback.
* **Apple Aesthetic:** Rounding border-radius follows Apple's squirclish `12px` to `16px` values.
