# MONI UI 5.0 Accessibility

This document outlines high contrast and keyboard optimizations:

## Key Operations

1. **High Contrast Mode**:
   - Toggles through the settings view to inject high contrast overrides, boosting label text readability against navy layouts.

2. **Keyboard Palette Navigation**:
   - `Ctrl + P` (or `⌘ K`) opens the Command Palette overlay immediately. Arrow selection maps standard action routes.

3. **Forms**:
   - Message composers support direct `Enter` submit commands and clear focus indicator outlines.
