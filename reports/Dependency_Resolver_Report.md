# Dependency Resolver Report

## Supported Ecosystems
`DependencyResolver` maps development dependencies to configuration files:
- **npm / pnpm / yarn**: Writes `package.json` package records.
- **pip / poetry**: Plans `requirements.txt` or `pyproject.toml` listings.
- **cargo**: Prepares `Cargo.toml` dependency crates blocks.
- **gradle / maven**: Structures `build.gradle` or `pom.xml` build maps.
- **pub**: Outputs `pubspec.yaml` configurations.
- **nuget**: Generates `.csproj` package reference items.

- **Status**: Planning mode (No package download executions).
