# Navigation Report

## Navigation Layout Designs
Enforces standard layouts for navigators to prevent visual clutter:
- **Top Bar**: Displays breadcrumbs, system notifications, and theme controllers.
- **Left Rail**: Hosts vertical icons to swap active workspaces (Home, Studio, Builder, etc.).
- **Console Bar**: Bottom dock for logs, native bridge states, and bridge status.
- **Quick Action Bar**: Floating palette for palette access.

- **Status**: Responsive navigators declared.
- **Accessibility**: ARIA labels mapped.
