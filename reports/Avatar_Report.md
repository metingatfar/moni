# Avatar Report

## Assistant Avatar Specification
The MONI Assistant Avatar acts as a conversational UI hub in the bottom-right corner:
- **Greeting Cards**: Greets users dynamically depending on session parameters.
- **Task Milestones**: Congratulates users on completing compilation steps.
- **Guidance Drawer**: Explains validator warnings and tech recommendations.

- **Status**: Mock avatar panel integrated.
- **Personality Mode**: Friendly and informative.
