# MONI UI 5.0 — Final Design System Report

This report confirms the locked and frozen design language elements for the MONI application.

## 1. Locked Visual Style Assets

- **Theme Palette**: Deep midnight navy background theme (`#090a15` base, `#0b0d1b` navigation sidebar, `rgba(17, 19, 36, 0.7)` card overlay backgrounds).
- **Accent Highlight**: Purple violet color token (`#7c3aed` base, `#a78bfa` hover accent states).
- **Materiality**: Blur glass panels and buttons with thin border details.
- **Corner Metrics**: Clean rounded corners (`12px` to `16px`) and smooth shadows.

## 2. Desktop UI Layout Architecture

- **Header Row**: Logo badge, Search launch field, Executive Mode selector pill, Action tools (🔔, ⚙️), Language selector, and Avatar.
- **Left Navigation**: Collapsible menu lists, SOHBETLER second-column navigation lists, centered MONI Orb visual widget card, and User Profile badge.
- **Center Area**: Chat view workspace containing tabs, header indicators, message streams, action buttons, and text composer.
- **Right Sidebar Panel**: Active provider card, circular system health widgets (CPU, RAM, DISK), tools dashboard, and shortcut keys.
- **Bottom Drawer Dashboard**: Son Aktif Dosyalar, Günlük Özet, Aktif Görevler list, and Pomodoro timer.
- **Status Bar**: Mode, Gemini 1.5 Pro, Online status, Türkçe, v5.0.0, CPU, RAM, and time.
