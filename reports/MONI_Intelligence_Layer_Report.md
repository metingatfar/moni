# MONI 3.5 — Intelligence Layer Architecture

## 1. Overview
The Intelligence Layer acts as a central hub managing context awareness, user timeline logging, and notifications for the MONI AI Operating System.

## 2. Core Components
* **ActivityTimeline:** Logs critical project, database, and settings events.
* **NotificationCenter:** Exposes notification listings and read statuses.
* **Context Manager:** Interacts with the layout context to serve contextual LLM prompts.
