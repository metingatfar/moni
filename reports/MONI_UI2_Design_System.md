# MONI 2.0 UI/UX Design System Specification

## 1. Core Principles
* **Apple & Cursor Quality:** Dark-space glassmorphism, precise spacing, and rich visual styling.
* **Calm Tech Experience:** Smooth transitions, non-disruptive alerts, and ambient neon glow.
* **High Contrast & Accessible:** Vibrant accents on rich dark space surface background.

## 2. Color Palette Tokens
| Variable | Value | Purpose |
| --- | --- | --- |
| `--bg-base` | `#0B0F17` | Deep cosmic dark background |
| `--bg-surface` | `rgba(17, 24, 39, 0.65)` | Translucent glass surface |
| `--bg-surface-elevated` | `rgba(31, 41, 55, 0.75)` | Hover cards and popups |
| `--accent-cyan` | `#00f0ff` | Primary action & glow |
| `--accent-purple` | `#9d4edd` | Assistant branding |
| `--accent-green` | `#10b981` | Success / online state |
| `--accent-red` | `#ef4444` | Error / offline state |

## 3. Typography Hierarchy
* **Sans Font:** `Outfit`, `Plus Jakarta Sans`, system-ui.
* **Mono Font:** `JetBrains Mono`.
* **Titles:** Heavy font-weight (800) with letter spacing.
* **Body:** Clean tracking, highly legible leading (1.6).

## 4. Interaction Tokens
* **Glass blur:** `20px` blur filter.
* **Shadow:** `0 8px 32px 0 rgba(0, 0, 0, 0.5)`.
* **Transitions:** `0.3s cubic-bezier(0.16, 1, 0.3, 1)` ease-out.
