# MONI 3.5 — Smart Recommendation Engine Report

## 1. Recommendation Engine Logic
Scans user states to output context-driven actions:
* **Tasks pending:** Suggests checking off uncompleted items.
* **Project loaded:** Recommends opening or completing work on the directory.
* **Inactivity:** Periodically prompts checking workspace health.

## 2. Frequency Control
* Low, Normal, and High frequency options allow configuring check intervals.
