# MONI Studio Report

* **Overview**: MONI Studio transforms the platform into a unified AI Software Engineering IDE.
* **Architecture**: Integrates file explorer, text editor, terminal console, and live browser previews.
* **State Management**: Governed by `StudioWorkspace` which coordinates active tabs, cursor positions, and problems panel.
* **Security & Constraints**: Remains a planning/mock workspace; does not modify user's workspace without explicit consent.
