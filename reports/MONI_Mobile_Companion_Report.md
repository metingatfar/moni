# MONI 3.0 Mobile AI Companion Interface

## 1. Interface Grid
Adapts below 768px to render a card layout:
* Minimal top title bar.
* Centered radial visualizer Orb.
* Grid buttons: Konuş, Yaz, Projeler, Bellek, Görevler, Ayarlar.
* Bottom navigation tabs.

## 2. Full-Screen Voice Assistant
Clicking "Talk" launches the dedicated fullscreen screen containing the glowing Orb, streaming transcription log blocks, and audio stop controls.
